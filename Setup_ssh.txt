1- Create public and private key pair.
    ssh-keygen -t ed25519 -C "bmorta01@student.ubc.ca"

2- Copy the SSH public key to your clipboard.
    clip < /c/Users/LBRY-SVC-Patron/.ssh/id_ed25519.pub
3- In the upper-right corner of any page on GitHub, click your profile picture, then click  Settings.
4- In the "Access" section of the sidebar, click  "SSH and GPG keys".
5- Click New SSH key or Add SSH key.
6- In the "Title" field, add a descriptive label for the new key. 
    For example, if you're using a personal laptop, you might call this key "Personal laptop".
    Enter "UBC Library Public Computer".
7- Select the type of key, either authentication or signing. 
    For more information about commit signing, see "About commit signature verification".
    Select "Authentication Key".
8- Click Add SSH key.
9- If prompted, confirm access to your account on GitHub. For more information, see "Sudo mode".

10- Start ssh-agent in the background 
    eval "$(ssh-agent -s)"
11- Add your public ssh key for Github and Github passphrase 
    ssh-add /c/Users/LBRY-SVC-Patron/.ssh/id_ed25519

        > Identity added: /c/Users/LBRY-SVC-Patron/.ssh/id_ed25519 (bmorta01@student.ubc.ca)
12- Test Github ssh connection
